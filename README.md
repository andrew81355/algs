Goal
The goal is to model part of Heidelberg, Germany as a weighted graph and
find the shortest path from different locations to the nearest hospital.
 
Algorithms
- Dijkstra
- A*
 
City
Heidelberg, Germany
 
Hospitals used
- Universitätsklinikum Heidelberg
- St. Josefskrankenhaus Heidelberg
- ATOS Klinik Heidelberg
- Krankenhaus Salem
 
How to run
python project.py
